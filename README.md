OmniTracker-Pro allows you to monitor any progress in statistics.

It's a web-app, and can run on any browser/device. It can be easily installed on Android and Apple mobile devices.

Priorities and notes:
1) Local first, very limited and non-repeating internet server connections. (Only when run for the first time)
2) Checks for version updates only weekly.
3) No data is stored on any servers. All data is saved on local devices. They are not encrypted, so make sure your device is protected!
4) The user-data can be backed up using the export feature. It's up to the user to protect the exported data.
5) It doesn't sync across devices. Because it runs totally local. Device changes must be manually handled by the user (e.g. all stats must be exported, and imported (if and when that feature is available))
6) Developed (prompted) and briefly tested by a human, using chat-bot AI (Gemini).

## How to Use This Library

### For Habit Tracker App Implementation:
1. **Categorize for Discovery**: Group habits by life area (Health, Mind, Productivity, etc.)
2. **Search & Filter**: Allow users to search by keyword (#energy, #quick, #morning)
3. **Popular/Featured**: Highlight most-selected habits
4. **Challenge Packs**: Curated sets (e.g., "Morning Routine Pack": Wake Early, Morning Sun, Meditation, Healthy Breakfast)
5. **Customization**: When a user selects a habit, pre-fill but allow immediate editing of both name and unit

### For Personal Use:
1. **Start Small**: Choose 1-3 habits to build consistency
2. **Stack Habits**: Attach new habits to existing routines
3. **Review Weekly**: Adjust what's not working
4. **Celebrate Streaks**: Use the tracker's statistics to maintain motivation
5. **Customize Units**: Change generic units to what works for you (e.g., "pages" instead of "minutes" for reading)
